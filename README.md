# Windows colorschemes for Neovim

---

## Install

```lua
use "woife5/windows.nvim"
```

Using lazy.nvim:

```lua
{
    "woife5/windows.nvim",
    config = function()
        -- Activation
        vim.cmd[[colorscheme windows-xp]]
    end
}
```

## Colorschemes

-   `windows-xp`
-   `windows-classic`
